## üåä What is Zenclora?

**Zenclora** is a Debian-based desktop Linux distribution optimized for stability, performance, and daily use. It features a customized, minimalist GNOME interface designed to be easy on the eyes while remaining powerful.

Our goal is to eliminate complex Linux usage. No need to get lost searching for repositories or dealing with complex configurations. Zenclora works right out of the box.

## ‚ö° The Zenclora Difference: Smart Commands

Zenclora includes special automation commands to save you hours of setup time. No more hunting for repos!

| Command | Description |
| :--- | :--- |
| `sudo update` | Update the system quickly and safely. |
| `install-flatpak` | Set up Flatpak support instantly. |
| `install-steam` | Install Steam for gaming. |
| `install-vibecoding` | Install AI IDEs (Cursor, Windsurf, AntiGravity). |
| `install-liquorix` | Install the high-performance Liquorix Kernel. |
| `install-officesuite` | Get your office productivity tools. |
| `zenclora-fetch` | Display system information. |

*(See the full list in [Documentation](https://zenclora.org/docs.html))*
